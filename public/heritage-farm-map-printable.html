Critical Hunting Rules & Safety

DOWNED GAME PROTOCOL: If you down game and it runs onto neighboring property (outside of property controlled by M77 AG), you MUST email hunting@m77ag.com immediately before pursuing. Emergency contact provided in booking confirmation. Do NOT enter neighboring property without authorization. NO RESPONSE DOES NOT MEAN PERMISSION.

‚ö†Ô∏è IMPORTANT REMINDERS

  • All hunters must have valid Colorado hunting licenses and appropriate tags
  • Email hunting@m77ag.com for any downed game issues (emergency contact in booking confirmation)
  • Respect property boundaries at all times - GPS can have 30-50 foot errors
  • Check game rest periods before hunting each parcel
  • Pack out what you pack in - leave no trace
  • Close all gates behind you
  • You have exclusive access to the property for your booked dates
  • Vehicle dash card must be visible in windshield while hunting
  • No shooting within 50 yards of buildings, roads, or property boundaries

GPS ACCURACY WARNING: Cell phone GPS can have 30-50 foot errors. Always stay well inside the boundary lines. When in doubt, move toward the CENTER of the property. Walk the boundaries when you first arrive to familiarize yourself with the terrain and landmarks.

PROPERTY BOUNDARIES - CRIMINAL TRESPASSING WARNING: Stay INSIDE property boundaries at all times. Trespassing on adjacent land will result in immediate loss of hunting privileges and criminal prosecution. How to View Property Boundaries

OPTION 1: OnX Hunt App (Recommended for Field Use)

  1. Download OnX Hunt app from App Store or Google Play
  2. Download the Heritage-Farm-Boundaries.kml file to your phone
  3. Open OnX Hunt app and tap Menu (‚ò∞) > Waypoints > Import
  4. Select the KML file from your downloads
  5. Property boundaries will display on your map with real-time GPS tracking

Note: OnX requires a subscription but is the best tool for hunters. Download at onxmaps.com

OPTION 2: Google Earth (Desktop)

  1. Download the Heritage-Farm-Boundaries.kml file
  2. Open Google Earth Pro (free at earth.google.com)
  3. Go to File > Open and select the KML file
  4. Expand "Heritage Farm" in the left sidebar to see all parcels
  5. Double-click any parcel name to zoom to that location

OPTION 3: Google Earth Mobile App

  1. Download Google Earth app from App Store or Google Play
  2. Download the KML file to your phone
  3. Open file manager and tap the KML file
  4. Select "Open with Google Earth"
  5. View parcels under "Projects" menu
