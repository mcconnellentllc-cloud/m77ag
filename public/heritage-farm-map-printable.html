How to View Property Boundaries

OPTION 1: OnX Hunt App (Recommended for Field Use)

  1. Download OnX Hunt app from App Store or Google Play
  2. Download the Heritage-Farm-Boundaries.kml file to your phone
  3. Open OnX Hunt app and tap Menu (‚ò∞) > Waypoints > Import
  4. Select the KML file from your downloads
  5. Property boundaries will display on your map with real-time GPS tracking

Note: OnX requires a subscription but is the best tool for hunters. Download at onxmaps.com

OPTION 2: Google Earth (Desktop)

  1. Download the Heritage-Farm-Boundaries.kml file
  2. Open Google Earth Pro (free at earth.google.com)
  3. Go to File > Open and select the KML file
  4. Expand "Heritage Farm" in the left sidebar to see all parcels
  5. Double-click any parcel name to zoom to that location

OPTION 3: Google Earth Mobile App

  1. Download Google Earth app from App Store or Google Play
  2. Download the KML file to your phone
  3. Open file manager and tap the KML file
  4. Select "Open with Google Earth"
  5. View parcels under "Projects" menu
