1. Control Valve Calibration

The Valve Response Number tunes how the control valve reacts to the difference between actual rate and target rate. It does not, by itself, change how many gallons per acre come out the tips.

The short answer on higher vs. lower

Higher Valve Response Number = valve moves faster and more aggressively. Lower number = valve moves slower.

Neither direction adds spray volume by itself. If the rate is chasing (bouncing above and below target), lower the response. If the rate is slow to catch up when speed changes, raise it.

If the sprayer is under-applying gallons per acre, this is almost never the valve response number. Skip to Section 2 (Flowmeter Calibration). That is the number that changes actual output.

Where to find it on the 2630

Menu path: Menu → GreenStar → Rate Controller → Setup → Valve Cal

The Valve Response Number is a four-digit code. Each digit tunes a different behavior of the servo valve:

Digit 1
Valve backlash / dead time. Higher = more anticipation for a sluggish valve.
Digit 2
Valve response speed. Higher = faster valve movement per correction cycle.
Digit 3
Response deadband. Higher = valve ignores small errors (steadier, less hunting).
Digit 4
Fine adjust on deadband. Same idea as digit 3, finer step.

Common starting points for a servo control valve on this class of Ag Chem floater:

SymptomTry
No idea, starting fresh0743 (JD default for standard servo valve)
Rate is slow to reach target after a speed changeRaise digit 2 (e.g. 0743 to 0843)
Rate hunts around target (bounces high/low)Lower digit 2, or raise digit 3 (e.g. 0743 to 0753)
Slight lag at startup onlyRaise digit 1 (e.g. 0743 to 1743)

Change one digit at a time, run a pass, watch the actual rate on the display, then re-tune.

2. Flowmeter Calibration — the number that changes spray volume

If a catch test at the tips shows less than the target GPA, this is the setting to fix. It tells the 2630 how many pulses the flowmeter sends per gallon.

Higher flowmeter calibration number = MORE actual spray output. Lower cal number = less actual output.

Because the display divides incoming pulses by the cal number to get gallons, a higher cal number makes the display under-report the flow it is seeing. The rate controller then opens the valve wider to hit target rate, so more product actually goes out the tips.

The Terra Gator is currently under-applying, so the fix goes UP. Raise the flowmeter cal number in small steps and re-catch-test between each change.

Where to find it on the 2630

Menu path: Menu → GreenStar → Rate Controller → Setup → Flowmeter Cal

The value stamped on the plate on top of the flowmeter is the factory pulses-per-gallon number. It is a starting point only. Every flowmeter drifts with wear, chemical residue, and screen condition, and the direction of the drift is almost always the same: worn turbines spin faster than they should, which means the cal needs to go up over time to match reality.

How to correct it with a catch test

  1. Fill the tank with clean water. Park on level ground. Bring the pump up to normal operating pressure.
  2. On the 2630, note the current Flowmeter Cal Number. Write it down before changing anything.
  3. Set a target rate (for example 15 GPA) and drive one measured pass, or run stationary with all booms open into calibrated buckets under each tip.
  4. Compare the actual gallons applied to what the 2630 says was applied.
    • Display says 15 GPA, tips actually put out 12 GPA → sprayer is under-applying (our current problem).
    • Display says 15 GPA, tips actually put out 18 GPA → sprayer is over-applying.
  5. Correct the cal number with this formula:

    New Cal = Old Cal × (Display Gallons ÷ Actual Gallons)

  6. Enter the new number. Repeat the catch test to verify. Two rounds of correction is normal.

Worked example for our current problem

The Terra Gator is short on spray volume. Say the flowmeter plate reads 720 pulses per gallon and a catch test shows the display claims 15 GPA but the tips are really putting out 12 GPA:

New Cal = 720 × (15 ÷ 12) = 720 × 1.25 = 900

The cal number goes from 720 up to 900. Enter 900 on the 2630, run the catch test again at the same target rate, and expect the tips to now put out very close to 15 GPA. If it still comes up short, apply the same formula against the new numbers and step it up again.

Sanity check. If the correction says the cal needs to move more than about 20% in one step, stop and inspect the flowmeter itself before entering the new number. A large correction usually means the meter is plugged, worn, or has a stuck turbine — not that the number was that far off.

Rule out these first, before touching the cal number

  • Plugged flowmeter screen or chemical build-up on the turbine. Pull the meter, clean the screen and rotor, spin the turbine by hand — it should spin freely.
  • Worn or wrong nozzle tips. A worn tip flows more, not less — but a mismatched replacement can flow less than expected. Check tip color/number against the rate chart for our target speed and pressure.
  • Pressure sitting at the low end of the tip range. Verify pump pressure against the tip chart for the current GPA and MPH.
  • A boom section not fully opening, a partially closed manual valve, or air in the plumbing.
  • Speed sensor calibration off — the 2630 uses ground speed to calculate GPA. If speed reads high, gallons per acre will read high while real output is low.

3. Implement and Machine Setup

Before any calibration will hold, the 2630 has to know it is driving the Terra Gator floater and the sprayer boom, not a planter or a spreader.

Machine profile

  1. Power on the 2630 with the key in the run position and the master boom switch OFF.
  2. Menu → GreenStar → Equipment → select or create the Terra Gator machine profile.
  3. Enter GPS receiver offsets (StarFire ITC, Unit #807). Antenna is at the cab roof center. Measure fore/aft from the boom.
  4. Enter boom width. Number of sections. Section widths. Nozzle spacing.
  5. Set implement type to Self-Propelled Sprayer.

Speed source

  • Primary: GPS ground speed from the StarFire receiver.
  • Backup: radar or wheel speed if available. Set the failover threshold so a bad GPS fix does not zero out rate control mid-pass.

4. Product, Rate, and Boom Setup

This is where the target GPA lives, and where boom section behavior is defined.

Product setup

  1. Menu → GreenStar → Rate Controller → Products.
  2. Create a product. Name it plainly (Roundup, Liberty, Fertilizer 32-0-0, and so on).
  3. Set the target application rate. Units gallons per acre.
  4. Set the minimum and maximum pressure. These are the guardrails the valve will not cross even if it is chasing rate.

Boom sections

  1. Confirm section count matches the physical boom.
  2. Assign each section switch on the console to the correct physical section. Walk the boom with the switches and check.
  3. Set Look-Ahead (turn-on and turn-off) so sections come on right at the headland edge. Start at 0.5 seconds and adjust.
  4. Enable Boom Section Control (BSC) for auto shut-off in overlap and outside boundaries. Requires a coverage map or field boundary.

Priming

Before the first pass, prime the booms: open every section, run the pump at operating pressure, and watch every tip for a solid pattern. A single plugged tip will drop total output enough to look like a calibration problem.

5. Field Check and Troubleshooting

Pre-application checklist

  • Tank agitation on before adding product.
  • Fresh water rinse of the flowmeter after every chemical change.
  • Speed on the 2630 matches the speedometer within 0.2 MPH.
  • Rate on the 2630 stabilizes at target within 3 seconds of opening booms.
  • Pressure gauge on the boom matches the tip chart for target GPA at target MPH.

Common symptoms and where to look

SymptomMost likely cause
Actual GPA at the tips is below target (current problem)Flowmeter cal number set too low, or plugged flowmeter screen, or nozzles too small for the target rate/speed combo. Raise the cal number per Section 2.
Rate on display bounces above and below targetValve response number too aggressive. Lower digit 2 or raise the deadband digits.
Rate takes 5+ seconds to reach target after opening boomsValve response too slow. Raise digit 2. Also check pump pressure is up before opening.
Rate reads correct on 2630 but tips are dry on one sectionSolenoid, wiring, or fuse for that section. Not a calibration issue.
2630 shows a Pressure Low faultProduct tank low, plugged strainer, agitation valve open too far, or pump not primed.
Coverage map has gaps or double-application stripesLook-ahead turn-on/turn-off timing, or GPS receiver offsets entered wrong.

Record what you change. Write down the old cal number, the new cal number, the date, and the catch-test result each time. The 2630 does not keep a change log. When something drifts next season, that notebook is what tells you where to start.
